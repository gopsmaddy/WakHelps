WHAT IS RSA Keys Components? 
Users E, D and P*Q to encryp or decrypt information

HOW TO ENCRYPT? 
1.RSA Public Key Exponet and RSA MODULUS of Receiver with clear data. Then encrypted data which only can be decrypted using Receivers private key (use for data/key transport as asymetric way)
2.RSA Private Key Exponet and RSA MODULUS of Sender with clear data. Then encrypted data which can be decrypted any one who has Senders public key (Users in a certificate to create signatures)

HOW TO DECRYPT?
1. Encrypted data with RSA MODULUS and RSA Public Key Exponet (Senders)
2. Encrypted data with RSA MODULUS and RSA Private Key Exponet (Receivers)
