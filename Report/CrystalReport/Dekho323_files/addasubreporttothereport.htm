Add

1.   To create a report, click Add.

2.   Enter the Report Name.

3.   Select the report type.

4.   Select Report Template. If you selected Crystal Reports in the report type, the options here will be all rpt documents found in the layout directory.  You can change you layout directory by setting the ‘layout.dir’. See the Settings Section of this document.

5.   Click Next to continue.

6.   Select the maps on which the report will be available.  Use the Select All and Deselect All buttons or tick the boxes individually.

Note: the report must also be added to the roles that a user is a member of in order for the user to see the report in Dekho.

7.   Click Next to continue.

8.   Select the Roles which will have access to the Report.  Use the Select All and Deselect All buttons or tick the boxes individually.

Note: the report must also be added to the maps that the user is viewing for the report to appear in Dekho.

 

9.   Click Save and Close.

10.  The Report has been successfully created.
